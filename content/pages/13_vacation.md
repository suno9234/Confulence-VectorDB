# 휴가 규정

## 연차 유급휴가

| 근속 기간 | 연차 일수 |
|-----------|-----------|
| 1년 미만 | 매월 1일 발생 (최대 11일) |
| 1년 이상 | 15일 |
| 3년 이상 | 16일 |
| 5년 이상 | 17일 |
| 7년 이상 | 18일 |
| 9년 이상 | 20일 (최대) |

## 특별 유급휴가 (법정 외 사내 제도)

| 종류 | 일수 | 비고 |
|------|------|------|
| 창립기념일 | 1일 | 매년 3월 15일 (주말이면 직전 금요일) |
| 생일 휴가 | 1일 | 생일 당일 또는 ±3일 이내 사용 |
| 결혼 휴가 | 5일 | 결혼식 전후 1개월 이내 사용 |
| 배우자 출산 | 10일 | 출산일로부터 90일 이내 사용 |
| 경조사 휴가 | 1~5일 | 경조사 유형별 상이 (아래 표 참고) |
| 리프레시 휴가 | 3일 | 근속 3년 이상, 매년 1회 |

## 경조사 휴가 세부 기준

| 사유 | 일수 |
|------|------|
| 본인 결혼 | 5일 |
| 자녀 결혼 | 1일 |
| 부모 회갑/칠순 | 1일 |
| 부모 사망 | 5일 |
| 배우자 사망 | 5일 |
| 자녀 사망 | 3일 |
| 형제자매 사망 | 1일 |
| 조부모 사망 | 2일 |

## 휴가 신청 방법

1. **카카오워크 앱** → 근태관리 → 휴가 신청
2. 1일 이상 연차: **3일 전** 신청 필수
3. 당일 긴급 병가: 09:30 이전 팀장에게 슬랙/전화 고지 후 사후 신청
4. 경조사 휴가: 증빙 서류를 인사팀(hr@nexusai.co.kr)에 5일 이내 제출

## 연차 소진 권고

- 연차 미사용 시 다음 연도 이월 불가 (법정 기준)
- 12월 1일 기준 잔여 연차 5일 초과 시 인사팀에서 소진 권고 메일 발송
- 잔여 연차는 연도 말 수당 미지급 (법정 연차 보상의무 대상 제외 시)
