# 재택근무 정책

최종 개정: 2024년 9월 1일 / 담당: 인사팀 오민지

## 기본 원칙

- **주 2회 재택** 가능 (월~금 중 자율 선택)
- **수요일은 전사 출근일**: 예외 없이 사무실 출근 필수
- **팀별 추가 출근일**: 팀장이 추가 출근일을 지정할 수 있음 (월 최대 1일)
- 고객사 방문, 중요 회의가 있는 날은 출근 필수

## 재택근무 신청 방법

1. 전날 퇴근 전까지 카카오워크 앱에서 '재택근무' 신청
2. 팀장 자동 알림 발송 → 팀장이 24시간 내 승인/반려
3. 승인 후 당일 재택 출근 시 앱 GPS 인증으로 출근 등록

## 재택 근무 환경 지원

- **모니터**: 1대 대여 가능 (IT지원팀 신청, helpdesk@nexusai.co.kr)
- **인터넷 비용**: 월 3만 원 보조 (재택 근무일 10일 이상 시 자동 지급)
- **장비 구매 지원**: 마우스, 키보드 등 소모품 연 15만 원 한도 지원

## 보안 규칙 (재택 시 필수)

- VPN 접속 필수 (내부 시스템 접근 시): `vpn.nexusai.internal`
- 공용 Wi-Fi 사용 금지 (카페, 도서관 등)
- 화면 녹화/스크린샷 공유 시 고객 데이터 노출 주의
- 노트북 자리 비울 때 잠금 화면 활성화 필수

## 재택 불가 사유

아래 상황 발생 시 팀장 판단 하에 출근 전환 가능:
- 연속 3일 이상 재택으로 팀 협업 저해 우려
- 수습 기간(3개월) 중인 구성원 (수습 중 재택 불가)
- 인사위원회 결정으로 재택 제한 중인 경우
