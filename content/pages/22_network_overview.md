# 네트워크 & 인프라 개요

## 네트워크 구조 요약

넥서스AI의 네트워크는 크게 **사무실 내부망**, **VPN 터널망**, **AWS 클라우드망** 세 영역으로 구성됩니다.

```
인터넷
  │
  ├── 사무실 내부망 (192.168.1.0/24)
  │     └── 직원 PC, 프린터, NAS
  │
  ├── VPN 터널 (10.0.0.0/8)
  │     └── 재택 직원 → 사내 시스템 접근
  │
  └── AWS (ap-northeast-2, 서울 리전)
        ├── VPC: 10.10.0.0/16
        ├── Public Subnet: 10.10.1.0/24 (Load Balancer)
        ├── Private Subnet: 10.10.2.0/24 (App Servers)
        └── DB Subnet: 10.10.3.0/24 (RDS)
```

## DNS 구성

- **내부 도메인**: `*.nexusai.internal` (사내 DNS 서버: 192.168.1.5)
- **외부 도메인**: `*.nexusai.co.kr` (Route 53 관리)
- VPN 연결 시 내부 DNS 자동 적용

## 인터넷 회선

- **주회선**: KT 기업용 광랜 1Gbps (강남파이낸스센터 빌딩 공급)
- **보조회선**: SKB 500Mbps (장애 대비 이중화)
- **회선 전환**: 주회선 장애 시 방화벽 자동 페일오버 (약 30초 소요)
